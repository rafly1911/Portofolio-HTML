Proyek ini dibuat sebagai bagian dari tugas mata kuliah Sistem Basis Data pada semester 4. Dalam proyek ini, kami berkolaborasi sebagai tim untuk merancang dan mengembangkan basis data untuk sebuah toko bunga online. Toko ini dirancang untuk menjual berbagai jenis bunga dan melayani berbagai kebutuhan pelanggan secara daring.

Sebagai bagian dari tim, tanggung jawab utama saya adalah merancang UI/UX atau antarmuka pengguna dari website. Saya bertanggung jawab penuh dalam mendesain tampilan dan interaksi yang ramah pengguna, memastikan bahwa pengalaman pengguna berjalan dengan lancar dan intuitif. Mulai dari pembuatan wireframe hingga desain akhir, saya memastikan setiap elemen visual dan navigasi berfungsi dengan baik untuk mendukung tujuan bisnis toko bunga ini.

Proyek ini merupakan Tugas Akhir untuk mata kuliah Pemrograman Berbasis Web di semester 5, yang dilaksanakan secara tim. Dalam proyek ini, saya bertanggung jawab penuh atas desain UI/UX, mencakup seluruh antarmuka dari halaman login hingga proses pengalihan pengguna untuk membeli tiket di website resmi event yang disediakan.
Website ini dirancang sebagai platform perantara bagi vendor pelaksana event untuk memasarkan acara mereka. Meskipun website ini tidak menangani transaksi langsung, pengguna diarahkan ke situs resmi event untuk pembelian tiket. Konten utama website meliputi informasi terkait event dan artikel-artikel yang relevan.
Peran saya dalam proyek ini melibatkan pembuatan desain yang ramah pengguna dan memastikan pengalaman navigasi yang intuitif, mulai dari akses awal hingga informasi yang disampaikan dengan jelas dan efektif.
